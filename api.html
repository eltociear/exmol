From given Examples, find best counterfactuals using readme_link:counterfactual generation This version works with regression, so that a counterfactual is if the given example is higher or lower than base. :type examples: List[Example] :param examples: Output from sample_space() :type delta: Union[Any, Tuple[float, float]] :param delta: float or tuple of hi/lo indicating margin for what is counterfactual :type nmols: int :param nmols: Desired number of molecules

sample_space(origin_smiles, f, batched=True, preset='medium', data=None, method_kwargs=None, num_samples=None, stoned_kwargs=None)

Sample chemical space around given SMILES This will evaluate the given function and run the run_stoned() function over chemical space around molecule. num_samples will be set to 3,000 by default if using STONED and 150 if using chemed. :type origin_smiles: str :param origin_smiles: starting SMILES :type f: Callable[[List[str], List[str]], List[float]] :param f: A function which takes in SMILES and SELFIES and returns predicted value. Assumed to work with lists of SMILES/SELFIES unless batched = False :type batched: bool :param batched: If f is batched :type preset: str :param preset: Can be wide, medium, or narrow. Determines how far across chemical space is sampled. Try “chemed” preset to only sample commerically available compounds. :type data: Optional[List[Union[str, Mol]]] :param data: If not None and preset is “custom” will use this data instead of generating new ones. :type method_kwargs: Optional[Dict] :param method_kwargs: More control over STONED, CHEMED and CUSTOM can be set here. See run_stoned(), run_chemed() and run_custom() :type num_samples: Optional[int] :param num_samples: Number of desired samples. Can be set in method_kwargs (overrides) or here. None means default for preset :type stoned_kwargs: Optional[Dict] :param stoned_kwargs: Backwards compatible alias for methods_kwargs :rtype: List[Example] :return: List of generated Example
