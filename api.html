Index of cluster, can be -1 for no cluster

Index relative to other examples

is_origin: bool = False¶

label: str = None¶

Label for this example

PCA projected position from similarity

Tanimoto similarity relative to base

plot_cf(exps, figure_kwargs=None, mol_size=(200, 200), mol_fontsize=10, nrows=None, ncols=None)¶

Draw the given set of Examples in a grid

  • ncols (Optional[int]) – number of columns to draw in grid

plot_space(examples, exps, figure_kwargs=None, mol_size=(200, 200), highlight_clusters=False, mol_fontsize=8, offset=0)¶

Plot chemical space around example and annotate given examples.

Parameters

  • examples (List[Example]) – Large list of :obj:Example which make-up points

  • exps (List[Example]) – Small list of :obj:Example which will be annotated

  • figure_kwargs (Optional[Dict]) – kwargs to pass to plt.figure

  • mol_size (Tuple[int, int]) – size of rdkit molecule rendering, in pixles

  • highlight_clusters (bool) – if True, cluster indices are rendered instead of :obj:Example.yhat

  • mol_fontsize (int) – minimum font size passed to rdkit

  • offset (int) – offset annotations to allow colorbar or other elements to fit into plot.

rcf_explain(examples, delta=(- 1, 1), nmols=4)¶

From given Examples, find best counterfactuals using readme_link:counterfactual generation

This version works with regression, so that a counterfactual is if the given example is higher or lower than base.

Parameters

  • examples (List[Example]) – Output from sample_space()

  • delta (Union[float, Tuple[float, float]]) – float or tuple of hi/lo indicating margin for what is counterfactual

  • nmols (int) – Desired number of molecules

Return type

List[Example]

sample_space(origin_smiles, f, batched=True, preset='medium', stoned_kwargs=None)¶

Sample chemical space around given SMILES

This will evaluate the given function and run the run_stoned() function over chemical space around molecule.

Parameters

  • origin_smiles (str) – starting SMILES

  • f (Union[Callable[[str, str], float], Callable[[List[str], List[str]], List[float]]]) – A function which takes in SMILES and SELFIES and returns predicted value. Assumed to work with lists of SMILES/SELFIES unless batched = False

  • batched (bool) – If f is batched

  • preset (str) – Can be wide, medium, or narrow. Determines how far across chemical space is sampled

  • stoned_kwargs (Optional[Dict]) – More control over STONED can be set here. See run_stoned()

Return type

List[Example]

Returns

List of generated Example
